Descriere

Aplicatia reprezinta un instrument web care ofera utilizatorului posibilitatea de a vizualiza pe harta, in timp-real, a imaginilor de pe retelele de socializare.

Imaginile sunt preluate de pe: 500px, Flickr si Instagram, ele continand informatii referitoare la localizarea geografica, cromatica, dimensiuni, orientare, tip, rezolutie, categorie etc.

Starea curenta a vizualizarii imaginilor (pe harta) se va putea salva local, cu posibilitatea restaurarii ulterioare.

Instrumentul va oferi suport pentru utilizarea diferitelor filtre de vizualizare referitoare la zone de interes (i.e. doar Europa centrala, numai imagini realizate in orase etc.), atribute ale fotografiilor, data si timp.

Tehnologii utilizate

  • HTML5
  • CSS3
  • JavaScript
  • Google Maps API
  • 500px API
  • Flickr API
  • Instagram API

Ghid de utilizare

Aplicatia web este disponibila online pentru a fi folosita de orice utiliator doritor

Adresa site-ului este: http://cliw-geos.s3-website-eu-west-1.amazonaws.com

Aplicatia pune la dispozitie 3 API-uri care permit afisarea pozelor pe harta: 500px, Flickr si Instragram.

Pentru 500px putem alege urmatoarele filtre:

  • Feature
  • Area
  • Category
  • Dimensions
  • Camera
  • Folcal length.

Introducand un cuvant/eticheta in caseta de text "Search photos" putem cauta poze dupa un anumit text.

In imaginea de mai jos avem pozele care contin in titlu/tag "Snowy" si sunt de pe 500px.

Flickr contine urmatoarele filtre:

  • Area
  • Location context
  • Taken date
  • Uploaded date
  • Dimensions

* Mai jos este prezentat un exemplu de cautare cu urmatoarele filtre: Flickr, Text - "Summer" Area - North America, Uploaded date - Today (sunt afisate pozele incarcate azi care contin textul "Summer", facute in America de Nord)
